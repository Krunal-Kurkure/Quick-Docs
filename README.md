# Easy Pdf

A React Native PDF Manager and PDF Creator app that helps users import, create, organize, edit, share, and delete PDF files with a smooth and simple interface.

## Overview

This app is designed to make PDF handling easy and efficient on mobile devices. Users can import PDFs from other apps, manage them inside the app, and create new PDFs from images using the built-in library feature.

The app includes a clean bottom tab navigation for fast movement between screens and a modern layout for viewing files in both grid and list formats.

## Features

### PDF Import

- Import PDF files into the app from other applications
- Keep all imported PDFs organized in one place

### PDF Management

- Rename PDF files
- Share PDF files
- Delete PDF files from within the app
- View PDF details such as:

  - File name
  - File size in MB
  - Date and time of creation

### PDF Viewer

- Open and read PDFs directly inside the app
- Display the PDF name in the header while viewing
- Simple and user-friendly reading experience

### Bottom Tab Navigation

- Smooth bottom tab navigation for easy switching between app sections
- Organized and accessible screen structure

### Library Tab

The Library tab allows users to create PDFs from images.

#### Image to PDF Creation

- Tap the `+` button to start creating a new PDF
- Pick images from the gallery
- Capture images using the camera
- Arrange images in the desired order
- Crop images before converting them
- Unselect images if needed
- Create the PDF instantly after selecting the images

### Home Screen

The Home screen provides a complete view of all PDFs stored in the app.

#### File Information Display

Each PDF card shows:

- PDF name
- File size in MB
- Date and time

#### Layout Options

- Grid view for a compact visual layout
- List view for a detailed file layout

## Key Highlights

- Fast and easy PDF importing
- Image-to-PDF creation
- In-app PDF preview
- Rename, share, and delete actions
- Grid and list file layouts
- Clean bottom tab navigation
- Smooth and organized user experience

## How It Works

1. Open the app
2. Import a PDF or create one from images
3. Manage the file using rename, share, or delete options
4. Open the PDF to read it inside the app
5. Switch between grid and list layouts as needed

## Permissions Required

Depending on device and Android version, the app may require:

- Storage access for reading and saving files
- Camera access for capturing images
- Gallery/media access for selecting images

## Tech Stack

- React Native
- React Navigation
- PDF viewing and PDF generation libraries
- Image picker and camera integration
- Native mobile UI components

## User Experience

The app is built with a simple, neat, and professional design so users can easily manage PDF files without confusion. Every feature is placed in a structured way to make navigation, file creation, and file reading comfortable and efficient.

## Conclusion

This React Native PDF app provides a complete solution for reading, creating, organizing, and managing PDF files in one place. It combines useful PDF tools with a clean mobile interface for a smooth user experience.
